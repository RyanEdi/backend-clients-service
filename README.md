# Clients Service

Servico responsavel pelo CRUD de clientes e periodos.

## Scripts

- npm run dev
- npm run build
- npm run start

## Variaveis de ambiente

- CLIENTS_SERVICE_PORT (default 3335)
- DB_HOST, DB_PORT, DB_USER, DB_PASSWORD, DB_NAME
- FRONTEND_URL

## Publicar como novo repositorio

1. git init
2. git add .
3. git commit -m "init clients-service"
4. git branch -M main
5. git remote add origin <URL_DO_REPO>
6. git push -u origin main
